# Starmap Desktop — Steam Store Page

## Title
Starmap Desktop

## Short Description
An Outer Wilds-inspired desktop star map that shows the real sky above you — switch between a world map wallpaper and an interactive celestial view oriented toward the Sun, planets, or the Milky Way.

## About This App

**Starmap Desktop** is a living desktop astronomy instrument. In everyday use it quietly displays a world map with real-time day/night shading. When you activate it, your desktop becomes a star field calibrated to your exact location on Earth.

### Features

- **Dual-mode wallpaper** — Passive world map (0–1 FPS) or active star map (60 FPS)
- **Real astronomy** — Planet positions powered by astronomy-engine (MIT), Hipparcos star catalog
- **Orient toward any body** — Face the Sun, Mars, Jupiter, or any solar system planet with unique particle effects
- **Milky Way view** — Zoom out to a galactic perspective centered on Earth
- **Outer Wilds aesthetic** — Warm campfire stargazing atmosphere, hand-painted cel-shaded planets
- **Steam Workshop** — Share and download particle themes, wallpapers, orientation presets, and soundscapes
- **Smart resource management** — Auto-pauses during fullscreen games and on battery power

### Controls

| Action | Input |
|--------|-------|
| Toggle mode | `Ctrl+Shift+S` or tray icon click |
| Rotate view | Mouse drag (active mode) |
| Zoom | Mouse wheel |
| Return to passive | `Esc` |
| Settings | Tray menu → 设置 |

**Recommended:**

- Memory: 8 GB RAM
