# Sidereal Desk v0.2 产品说明

## 定位

**Sidereal Desk（恒星时桌面仪）** 是一台桌面角落的天文仪器，不是壁纸、不是星图软件、不是番茄钟。

题材：**时间如何被看见** — 地方恒星时、暮光、月相，由真实历表驱动。

## 从 Starmap 迁移

| v0.1 Starmap | v0.2+ Sidereal Desk |
|---|---|
| WorkerW 壁纸 + 探索器双窗 | 单一透明浮窗 |
| Three.js 太阳系/星空 | SVG 多环钟面 |
| 壁纸引擎竞品红海 | Desktop Companion 蓝海 |
| 功能堆叠 | 单一乐器：看天 |

## 体验支柱

1. **钟面即答案** — LST 是主角；Minimal 模式下图例与首次提示降低学习成本
2. **暮光即氛围** — 面板色调随 twilight 相位变化
3. **札记即陪伴** — 每日一行观测记（i18n）

## v0.3.0-beta 范围

### In

- 透明可拖拽浮窗（Full / Compact / Minimal 三档）
- SVG 恒星时钟（LST 针、太阳标、日照弧、月相、0/6/12/18 时标）
- 地方恒星时 / 暮光 / 月相 / 日出日落
- 观测位置 + 海拔持久化；启动：saved > IP > default
- 系统托盘：关窗隐藏、左键 toggle、菜单 i18n
- 时间探索：滚轮 scrub、偏移显示、速度预设、步进、快捷键
- 设置内「如何读钟面」说明

### Out（后续版本）

- 每日收藏卡片导出
- macOS / Linux
- 深空探索模式

## 视觉

- 字体：DM Sans（数据）+ Instrument Serif（札记）
- 色：深靛底 + 烫金强调，随暮光微变
- 形式：圆环仪器，非仪表盘
