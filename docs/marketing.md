## Beta Testing Plan

### Phase 1 — Internal (Week 1–2)
- 5 testers: verify Windows wallpaper layer, astronomy positions
- Feedback form: Google Form or GitHub Issues

### Phase 2 — Closed Beta (Week 3–4)
- 20–50 testers via Steam Playtest (free)
- Test matrix:
  - Windows 10, 11 (22H2, 24H2)
  - macOS 13, 14, 15
  - Ubuntu 22.04/24.04 (X11 + Wayland)
- Key metrics: FPS in passive/active, CPU idle %, wallpaper layer stability

### Phase 3 — Open Beta (Week 5–6)
- Steam Next Fest demo (if timing aligns)
- Public issue tracker on GitHub
